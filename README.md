# LitecoinKit

`LitecoinKit` is a package that extends [BitcoinCore.Swift](https://github.com/curdicu/BitcoinCore.Swift) and makes it usable with `Litecoin` Mainnet and Testnet networks. 

## Usage

Because Litecoin uses Bitcoin's source code, the usage of this package has no differences from `BitcoinKit`. So please see [BitcoinKit.Swift](https://github.com/curdicu/BitcoinKit.Swift) for usage documentation

## Prerequisites

* Xcode 10.0+
* Swift 5+
* iOS 13+

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/curdicu/LitecoinKit.git", .upToNextMajor(from: "1.0.0"))
]
```
